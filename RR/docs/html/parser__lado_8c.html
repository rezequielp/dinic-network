static int parse_argument ( Lexer *  input,
u64 *  n 
)
static

Parsea un argumento de los ingresados.

Los caracteres que se lean deben ser dígitos para que el argumento sea válido.

Parámetros
inputEl analizador léxico del descriptor de entrada.
nEl argumento parseado resultante.
Precondición
input!=NULL.
Devuelve
Si no hubo error, asigna el argumento parseado en 'n' y retorna PARSER_OK.
Si hubo error, no asigna nada a 'n' y retorna PARSER_ERR.

static bstring next_bstring ( Lexer *  input,
int  flag,
const char *  str 
)
static

Lee el siguiente ítem.

Si flag=ACCEPT: Consume hasta encontrarse con un char distinto de 'str'. Si flag=EXCEPT: Consume hasta encontrarse con un char de 'str'.

Parámetros
inputEl analizador léxico del descriptor de entrada.
flagACCEPT|EXCEPT respecto al contenido de 'str'.
strLos caracteres con los que se compara cada caracter leído.
Precondición
input!=NULL, y flag debe ser una opción valida.
Devuelve
Un bstring del ítem leído. El llamador se encarga de liberarlo.
NULL caso contrario.

Lado parser_lado ( Lexer *  input)

Lee todo un Lado de input hasta llegar a un fin de línea o de archivo.

Se asegura que se consumió input hasta el primer error o hasta completar el lado y No se consumió ningún "\n".

Parámetros
inputEl analizador léxico del descriptor de entrada.
Precondición
input!=NULL.
Devuelve
Un lado bien formado si el parseo fue exitoso.
LadoNulo caso contrario.

int parser_nextLine ( Lexer *  input)

Consume el fin de línea.

Indica si encontró (o no) basura antes del fin de línea. Se asegura que no se consumió más entrada de la necesaria y el lexer esta detenido justo luego de un "\n" o en el fin de archivo.

Parámetros
inputEl analizador léxico del descriptor de entrada.
Precondición
input!=NULL.
Devuelve
PARSER_OK si no hay basura.
PARSER_ERR caso contrario.
