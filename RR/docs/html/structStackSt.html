Descripción detallada

Estructura principal de una pila.

Atención
Sobre una pila no se pueden correr dos iteraciones simultaneas ya que cada una movería el visor 'iter' accediendo a los elementos alternadamente. Esta característica es exclusiva. En futuros upgrades se evitara que se puedan correr diferentes iteraciones o se harán visores múltiples para que si se pueda.
