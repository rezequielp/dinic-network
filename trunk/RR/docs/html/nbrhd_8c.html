static void * findNbr ( Nbrhd  nbrs,
u64  y,
short int  dir 
)
static

Busca un vecino con nombre 'y', en la dirección especificada.

Parámetros
nbrsEl vecindario de 'x'.
yEl nombre del vecino.
dirDirección que se encuentra el vecino (lado FWD o BWD).
Precondición
'y' es vecino de 'x'.
Devuelve
La representación del lado dependiendo de la dirección en la que se encuentra 'y':
Si es forward, se retorna un puntero a un Fedge.
Si es backward, se retorna un puntero a un Bedge.

Genera el vinculo entre 'x' e 'y' (edge) convirtiéndolos en vecinos.

La relación es 'xy': 'y' vecino forward de 'x'; 'x' vecino backward de 'y'.

Precondición

int nbrhd_getFwd ( Nbrhd  nbrs,
int  rqst,
u64 *  y 
)

Busca el siguiente vecino forward.

La petición de búsqueda puede ser por el primer nodo (FST) de la tabla, o bien por el siguiente(NXT) del ultimo pedido. Si existe almacena el nombre en 'y'.

Nota
Tener en cuenta la documentación sobre las opciones de los parámetros. Verlo como un iterador de consultas a una tabla.
Parámetros
nbrsEl vecindario del nodo ancestro 'x'.
rqstSi se pide el primero 'FST' o un siguiente 'NXT'.
yVariable en la que se almacena el nombre del vecino encontrado.
Precondición
'nbrs' e 'y' no son nulos y 'rqst' es una opción valida (FST o NXT).
Devuelve
1 Si se encontró y se almacenó en 'y' un vecino.
0 Caso contrario.

Se aumenta el flujo para con el vecino 'y' por 'vf' cantidad.

Si 'y' es un vecino BWD, el valor del flujo se disminuye por 'vf' cantidad.

Parámetros
nbrsEl vecindario del nodo ancestro 'x'.
yEl nombre del vecino.
dirDirección que se encuentra el vecino (lado FWD o BWD).
vfEl valor de flujo.
Precondición
'y' es vecino de 'x'. 'vf' > 0.
Devuelve
Valor del nuevo flujo que se esta enviando entre 'x' e 'y'.
