Descripción detallada

Parseador de lados desde la entrada Estandar Input.

El path es un archivo con 3 int separados por un espacio entre ellos y luego un "\n". Esta estructura puede estar repetida varias veces. Al final hay un EOF. 

 s::= {'+int+' '+int+' '+int+'\n'}+EOF
 
 Ejemplo: 2 4 6\n2 4 54\n321321 321321 4888\nEOF

Lado parser_lado ( Lexer *  input)

Lee todo un Lado de input hasta llegar a un fin de línea o de archivo.

Se asegura que se consumió input hasta el primer error o hasta completar el lado y No se consumió ningún "\n".

Parámetros
inputEl analizador léxico del descriptor de entrada.
Precondición
input!=NULL.
Devuelve
Un lado bien formado si el parseo fue exitoso.
LadoNulo caso contrario.

int parser_nextLine ( Lexer *  input)

Consume el fin de línea.

Indica si encontró (o no) basura antes del fin de línea. Se asegura que no se consumió más entrada de la necesaria y el lexer esta detenido justo luego de un "\n" o en el fin de archivo.

Parámetros
inputEl analizador léxico del descriptor de entrada.
Precondición
input!=NULL.
Devuelve
PARSER_OK si no hay basura.
PARSER_ERR caso contrario.
