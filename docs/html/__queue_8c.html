int queue_destroy ( Queue  Q,
void **  garbage 
)

Destruye y libera la memoria de la cola Q preservando los elementos.

Esta función no elimina los elementos que se agregaron a la cola. Es tarea del llamador destruirlos si se consideraba necesario. Esto es así porque no se sabe cual es el tipo de las cosas encoladas y porque no es una tarea de queue_destroy.

Parámetros
QCola a destruir.
garbagePuntero a un arreglo de punteros de elementos que se agregaron a la cola. Si es NULL, queue_destroy no guardara los elementos en ningún lado. El llamador debe tener cuidado de no perder la referencia a esta memoria para no generar memory leaks.
Precondición
Q no es una cola nula.
Devuelve
Cantidad de elementos que se eliminaron efectivamente de la cola.
