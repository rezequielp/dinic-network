void print_dinicTime ( float  time)
static

Imprime el tiempo que tarda el algoritmo DINIC en ejecutarse en formato [hh:mm:ss.ms].

No contabiliza el tiempo de carga de los elementos del network ni otras acciones innecesarias. Esto es así porque se supone que el tiempo de carga varia mucho cuando se pide al sistema operativo interactuar con IO. De esta manera se logra un tiempo solamente de la parte del programa que implementa DINIC.

Parámetros
timeTiempo que tardo el algoritmo DINIC en correr medidos segundos.
